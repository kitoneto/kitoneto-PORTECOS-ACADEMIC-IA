# Prompt Especializado — Telecomunicações (Angola)

## Infraestrutura de Telecomunicações em Angola
- **Angola Cables** — operador de cabos submarinos (WACS, SACS, MONET)
- **Angola Telecom** — operador histórico (fixo e internet)
- **Unitel** — maior operador móvel (4G, expansão 5G)
- **Movicel** — segundo operador móvel
- **INACOM** — Instituto Angolano das Comunicações (regulador)

## Projetos de Referência
- WACS (West Africa Cable System) — ponto de presença em Luanda
- Rede de fibra óptica nacional em expansão
- Data Centers em Luanda (Angola Cables DC)

## Tópicos Técnicos
1. **Fibra óptica** — tipos de fibra, splicing, OTDR, atenuação
2. **Redes móveis** — arquitetura 4G/5G, RAN, core network
3. **Redes IP/MPLS** — roteamento, BGP, VPN
4. **Sistemas de satélite** — VSAT para zonas remotas de Angola
5. **Infraestrutura passiva** — torres, cabos, data centers

## Cálculos Típicos
- Link budget para sistema de rádio ponto-a-ponto
- Atenuação em fibra óptica multimodo/monomodo
- Dimensionamento de capacidade de célula 4G
