# Prompt Especializado — Inteligência Artificial (Angola)

## IA Aplicada ao Contexto Angolano
Angola tem um enorme potencial para IA aplicada a:
- **Setor petrolífero**: otimização de produção, manutenção preditiva
- **Agro-indústria**: previsão de colheitas, deteção de pragas
- **Saúde pública**: diagnóstico médico em zonas remotas
- **Infraestruturas**: monitorização de barragens e pontes
- **Finanças**: detecção de fraude, scoring de crédito

## Tópicos de Formação
1. **Machine Learning** — regressão, classificação, clustering
2. **Deep Learning** — redes neurais, CNN, RNN, Transformers
3. **NLP** — processamento de texto em português
4. **Computer Vision** — inspeção visual industrial
5. **MLOps** — deploy de modelos em produção
6. **IA Generativa** — LLMs, RAG, fine-tuning

## Problema Real: Previsão de Produção Petrolífera
Usar dados históricos de produção dos blocos offshore para:
- Prever declínio de produção (decline curve analysis com ML)
- Detecção de anomalias em equipamentos FPSO
- Otimização de programas de estimulação de poços

## Ferramentas e Frameworks
- Python (scikit-learn, TensorFlow, PyTorch)
- Jupyter Notebooks para análise exploratória
- Hugging Face para modelos de linguagem em português
- MLflow para gestão de experimentos
