# PORTECOS ACADEMIC IA — Painel Administrativo

## Descrição
Dashboard de gestão da plataforma PORTECOS ACADEMIC IA. Permite ao time de gestão monitorizar métricas, gerir utilizadores, cursos e conteúdos.

## Funcionalidades Planeadas
- [ ] Dashboard de métricas em tempo real
- [ ] Gestão de utilizadores (alunos, professores, admins)
- [ ] Gestão de cursos e conteúdos
- [ ] Relatórios de progresso dos alunos
- [ ] Gestão de certificados
- [ ] Configuração do Tutor IA
- [ ] Gestão de pagamentos e subscrições
- [ ] Logs de auditoria

## Tecnologias
- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- Integração com API REST

## Iniciar

```bash
cd apps/admin
npm run dev
# Abrir http://localhost:3001
```
