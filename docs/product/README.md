# PORTECOS ACADEMIC IA — Product Roadmap

## Visão do Produto
Tornar a formação técnica de qualidade acessível a todos os angolanos, independentemente da localização ou recursos financeiros, usando IA para personalizar a aprendizagem.

## Princípios
1. **Angola First** — tudo contextualizado para o mercado angolano
2. **Problemas Reais** — aprender resolvendo problemas do dia-a-dia profissional
3. **IA como Acelerador** — tutor disponível 24/7 em português
4. **Acessibilidade** — funciona em conexões lentas, com tier gratuito real

## Roadmap

## User Stories

### Como aluno...
- Quero escolher cursos pela área técnica para encontrar o que preciso
- Quero que o Tutor IA me explique passo a passo para não ficar bloqueado
- Quero um certificado para apresentar ao meu empregador

### Como empresa...
- Quero formar os meus técnicos em conteúdo específico do setor petrolífero
- Quero acompanhar o progresso da minha equipa
- Quero relatórios de competências para RH

### Como instrutor...
- Quero criar cursos facilmente sem saber programar
- Quero ver métricas de engajamento dos meus alunos
- Quero receber uma percentagem das subscrições geradas pelos meus cursos
