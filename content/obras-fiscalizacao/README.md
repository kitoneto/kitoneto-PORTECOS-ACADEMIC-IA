## 🔧 Problema Real — "Fiscalização de Betão Não Conforme em Luanda"

**Contexto:** Em obra de edifício de 8 andares em Talatona, o fiscal detetou betão aparentemente de fraca qualidade. Os cubos de controlo (150×150×150mm) ensaiados aos 28 dias deram:

| Cubo | Resistência (MPa) |
|------|-----------------|
| 1    | 24,8            |
| 2    | 23,1            |
| 3    | 27,2            |
| 4    | 21,5            |
| 5    | 25,4            |

**Betão especificado:** C25/30 (fck = 25 MPa)

**Pergunta:** O betão cumpre os requisitos normativos?

**Análise (EN 206):**
- Média dos 5 resultados = 24,4 MPa
- Valor mínimo = 21,5 MPa
- Para critério EN 206: média ≥ fck + 1,48σ = 25 + 1,48×2,1 ≈ 28,1 MPa

**Conclusão: NÃO CUMPRE.** A média é inferior ao critério normativo.

**Ações do Fiscal:**
1. Emitir Não-Conformidade formal por escrito
2. Notificar empreiteiro para apresentar plano de correção em 48h
3. Solicitar abertura de carotes para ensaio in situ
4. Eventualmente demolição e reconstrução dos elementos afetados

**Lição:** Em Angola, a fraude no betão é comum — adicionar excesso de água em obra. O fiscal deve estar presente nas betonagens e registar o slump test.
